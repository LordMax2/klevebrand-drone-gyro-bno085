# Klevebrand Drone Gyro – BNO085

BNO085 implementation for the [Klevebrand All-Drone Controller](https://github.com/LordMax2/klevebrand-all-drone-controller) abstract IMU/gyro interface.

- Lightweight I²C driver for BNO085
- Fused quaternion + raw gyro/accel/mag data
- Simple plug-and-play with the core library

Just include, initialize, and pass the instance to your controller.
